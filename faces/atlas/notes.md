# Atlas

Initial sample watch face for Milestone 1.

## Intent

Create a single prototype face that proves the early workflow:
- shared Pixel Watch frame
- shared time controls
- fixed/live preview modes
- explicit face metadata
- browser-first iteration surface

## Current shape

Atlas is an analog-hybrid face with:
- radial tick marks
- a central digital time readout
- date text
- hour/minute hands
- optional second hand in interactive mode
